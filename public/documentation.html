about different movies, directors, and genres.

Endpoint Map


Movies:

Endpoint: '/movies'

  • Method: GET
  • Description: Gets all Movies
  • Parameters: None
  • Data expected in request body:

    • None

  • Data expected in response:

    • A JSON object of all the movies

Endpoint: '/movies/:title'

  • Method: GET
  • Description: Gets movie by title
  • Parameters: title
  • Data expected in request body:

    • None

  • Data expected in response:

    • A JSON object a specific movie

Endpoint: '/movies/genre/:genreName'

  • Method: GET
  • Description: Gets genre of movie
  • Parameters: genreName
  • Data expected in request body:

    • None

  • Data expected in response:

    • A JSON object of a specific genre

Endpoint: '/movies/directors/:directorName'

  • Method: GET
  • Description: get data about a director
  • Parameters: directorName
  • Data expected in request body:

    • None

  • Data expected in response:

    • A JSON object of a specific director



Users:

Endpoint: '/users'

  • Method: POST
  • Description: Registers new user
  • Parameters: name
  • Data expected in request body:

    • JSON object of the new user: {"id": "", "name": "", "favoriteMovies": ""}

  • Data expected in response:

    • JSON object of the new user: {JSON object of the new user: {"id": "", "name": "", "favoriteMovies": ""}

Endpoint: '/users/:id'

  • Method: PUT
  • Description: Updates user information
  • Parameters:
    • id
    • name
  • Data expected in request body:

    • A JSON object of the new user, structured like: {"id": "", "name": "", "favoriteMovies": ""}

  • Data expected in response:

    • A JSON object of the new user, structured like: {"id": "", "name": "", "favoriteMovies": ""}

Endpoint: '/users/:id/:movieTitle'

  • Method: POST
  • Description: Adds movie to favoriteMovies
  • Parameters:
    • id
    • movieTitle
  • Data expected in request body:

    • None

  • Data expected in response:

    • A JSON of the user

Endpoint: '/users/:id/:movieTitle'

  • Method: DELETE
  • Description: Removes movie from favoriteMovies
  • Parameters:
    • id
    • movieTitle
  • Data expected in request body:

    • None

  • Data expected in response:

    • Text that shows the movie has been removed

Endpoint: '/users/:id/'

  • Method: DELETE
  • Description: Unregisters user
  • Parameters: id
  • Data expected in request body:

    • None

  • Data expected in response:

    • Text showing the user has been removed
